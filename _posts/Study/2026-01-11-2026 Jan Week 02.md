## 2026-01-06

### Chapter 03. 표준 입출력

**학습 내용**
- 표준 입출력 함수(`getchar`, `putchar`, `scanf`, `printf`)
- 버퍼 개념과 입력 버퍼 처리
- 형식 문자열과 서식 지정자

**주요 개념**
- `getchar()`: stdin 버퍼 사용, 엔터키 입력 필요
- `_getch()`: 버퍼 없이 즉시 입력 받음
- `scanf()` 사용 시 주의사항
  - 형식 문자열에 개행문자(`\n`) 포함 금지
  - 변수 주소 전달 필수 (`&` 연산자)
  - 입력 버퍼에 남은 개행문자 처리 (`%*c` 활용)

**실습 코드 하이라이트**
```c
// 입력 버퍼 개행문자 제거
scanf("%d%*c", &nAge);  // %*c로 개행문자 버림

// 실수 출력 서식
printf("%.2f\n", dData);  // 소수점 2자리까지
printf("%012.3f\n", 123.456);  // 12자리, 0으로 채움
```

**연습문제**
- 나이와 이름 입력받기 (버퍼 처리 포함)
- `getchar()`로 두 번째 문자만 출력

## 2026-01-07

### Chapter 04. 연산자 - 기본

**학습 내용**
- 산술 연산자와 형변환(Type Promotion)
- 대입 연산자와 l-value 개념
- 비트 연산자(AND, OR, XOR, NOT, Shift)
- 증감 연산자(prefix/postfix)

**주요 개념**
- **Type Promotion**: `char + int → int`, `int + double → double`
- **정수 나눗셈**: `5 / 2 = 2` (몫만 반환, 실수 변환 필요)
- **비트 마스킹**: 특정 비트 추출/설정/테스트
  ```c
  nData & 0x00FFFF00  // 중간 2바이트만 추출
  nData | 0x00080000  // 특정 비트 설정
  ```
- **증감 연산자**
  - Prefix(`++i`): 즉시 증가 후 사용
  - Postfix(`i++`): 사용 후 증가 (우선순위 최하)

**실습 예제**
- 두 정수 평균 계산 (형변환 활용)
- 초(second)를 시:분:초로 변환
- 값 교환(swap) 패턴

### Chapter 05. 연산자 - 응용

**학습 내용**
- `sizeof` 연산자
- 관계 연산자와 논리 연산자
- 삼항 연산자(`?:`)
- Short-circuit 평가

**주요 개념**
- **실수 비교 주의**: `300 == 299.99999`는 부정확할 수 있음
- **논리 연산 오류**
  ```c
  // 잘못된 예
  3 < nInput < 20  // (3 < nInput) < 20으로 평가됨
  
  // 올바른 예
  3 < nInput && nInput < 20
  ```
- **삼항 연산자**: `조건 ? 참값 : 거짓값`
- **Survival Search 패턴**: 최댓값/최솟값을 갱신하며 찾기

**실습 코드**
```c
// 최댓값 갱신 패턴
nMax = nInput > nMax ? nInput : nMax;
```

### Chapter 07. 반복문

**학습 내용**
- `while`, `for`, `do-while` 문
- 중첩 반복문
- `break`, `continue` 제어
- 반복문 내 변수 선언 주의사항

**주요 개념**
- **While vs For**
  - While: 조건 기반 반복
  - For: 횟수 기반 반복
- **반복문 내 변수**
  - 루프 외부 선언: 값 유지 (카운터, 누적기)
  - 루프 내부 선언: 매번 초기화 (주의!)
- **이중 반복문**: 구구단, 별 찍기 등

**실습 패턴**
```c
// 누적기 패턴
int nTotal = 0;
while (i < 5) {
    nTotal += aList[i];
    ++i;
}

// 중첩 반복문 (직각삼각형)
for (i = 0; i < 5; ++i) {
    for (j = 0; j <= i; ++j) {
        printf("*\t");
    }
    putchar('\n');
}
```

**연습문제**
- 4의 배수 개수와 총합
- 피라미드/트리 별 출력
- `goto` 문을 `do-while`로 변환

### Chapter 08. 배열

**학습 내용**
- 1차원 배열 선언과 초기화
- 배열과 반복문의 결합
- 문자열의 구조 (`'\0'` NULL 문자)
- 다차원 배열 (2차원, 3차원)

**주요 개념**
- **배열의 특징**
  - 배열 이름은 주소 상수 (l-value 불가)
  - 배열 복사는 요소별로 수행
  
  ```c
  // 잘못된 예
  aNewList = aList;  // 컴파일 에러
  
  // 올바른 예
  for (i = 0; i < 5; ++i)
      aNewList[i] = aList[i];
  ```
- **문자열**: 문자 배열 + NULL 종결자(`'\0'`)
  ```c
  char szBuffer[6] = {'H','e','l','l','o','\0'};
  char szData[8] = {"Hello"};  // 자동으로 \0 추가
  ```
- **2차원 배열**: `aList[행][열]`
  - 메모리상 행 우선 순서로 저장
  - 경계 초과 접근 주의 (`aList[0][4]`는 `aList[1][0]`과 동일)

**실습 예제**
- 5개 정수의 최댓값/최솟값 찾기
- 한글 문자열 길이 계산 (멀티바이트)
- 2차원 배열 행/열 합계 계산

### Chapter 09. 배열 프로그래밍 기법

**학습 내용**
- 정렬 알고리즘 (버블 정렬, 선택 정렬)
- 2차원 배열 응용 (지그재그, 달팽이 배열)
- Lookup 배열 활용

**주요 개념**
- **버블 정렬**: 인접한 요소 비교 & 교환
  ```c
  for (i = 4; i > 0; --i) {
      for (j = 0; j < i; ++j) {
          if (aList[j] > aList[j+1]) {
              // swap
          }
      }
  }
  ```
- **선택 정렬**: 최솟값 찾아서 앞으로 이동
- **지그재그 배열**: 홀수 행은 역방향 채우기
  ```c
  if (i % 2 == 0)
      for (j = 0; j < 5; ++j) list[i][j] = ++counter;
  else
      for (j = 4; j >= 0; --j) list[i][j] = ++counter;
  ```
- **달팽이 배열**: 방향 전환하며 나선형 채우기

**실습 예제**
- 열 방향 지그재그 배열
- 반시계 방향 달팽이 배열
- Lookup 테이블을 이용한 요금 계산

### Chapter 10. 함수 기본

**학습 내용**
- 사용자 정의 함수 선언과 정의
- 함수 설계 원칙 (UI와 기능 분리)
- 함수 원형(Prototype) 선언
- 전역 변수와 지역 변수
- 코드 재사용성과 의존성 관리

**주요 개념**
- **함수 구조**
  ```c
  반환형 함수명(매개변수) {
      // 함수 본문
      return 값;
  }
  ```
- **UI와 기능 분리**
  - 함수는 순수 기능만 담당
  - 입출력은 `main()`에서 처리
  ```c
  // 나쁜 예
  int GetFactorial(int n) {
      if (n < 1) {
          puts("ERROR");  // UI 코드
          return 0;
      }
      // ...
  }
  
  // 좋은 예
  int GetFactorial(int n) {
      if (n < 1) return -1;  // 에러 코드 반환
      // ...
  }
  ```
- **함수 원형 선언**: 컴파일러에게 함수 존재 알림
  ```c
  int Add(int a, int b);  // 원형 선언
  
  int main(void) {
      Add(3, 4);  // 사용 가능
  }
  
  int Add(int a, int b) {  // 정의
      return a + b;
  }
  ```
- **전역 변수 vs 지역 변수**
  - 전역 변수: 모든 함수에서 접근 가능, 남용 주의
  - 지역 변수: 함수/블록 내에서만 유효
  - **권장**: 매개변수로 데이터 전달 (의존성 최소화)

**설계 원칙**
1. 단일 책임: 함수는 하나의 기능만
2. 재사용성: 다양한 상황에서 사용 가능하게
3. 낮은 결합도: 전역 변수 대신 매개변수 활용

**실습 예제**
- 메뉴 기반 프로그램 구조
- 정렬 함수의 재사용 (배열 크기 매개변수화)
- 학점 계산 프로그램 (입력/처리/출력 분리)

**연습문제**
- `GetMin()` 함수: 4개 정수 중 최솟값 반환
- `GetFare()` 함수: 나이별 요금 계산
- 함수 원형 선언의 필요성 설명
